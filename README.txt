Berytus is a browser-based password manager governance framework 
for programmable authentication and registration sessions.

Head over to the proof of concept website to download the source
code or compiled binaries:

    https://poc.berytus.io

Berytus is detailed in a research paper that is accepted
and due to appear at the proceedings of the 2024 International
Conference on Information and Communications Security. The
ePrint is available on arXiv:

    https://arxiv.org/abs/2407.07205

We have a few housekeeping tasks that we need to take care of
before pushing the source code to this git repository; until 
then, the source code will remain accessible through the proof 
of concept website.
